# Temperature Converter

A simple web app that converts temperatures between Celsius and Fahrenheit.

## Usage

1. Enter a temperature value in the input field.
2. Select the unit you're converting **from** (Celsius or Fahrenheit).
3. Click **Convert** to see the result.

## How to Run

Open `lol.html` directly in a browser, or serve it locally:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080/lol.html`.
